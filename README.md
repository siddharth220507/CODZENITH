Student Marks Management and Analysis System

This project is a console-based C++ application designed to manage and analyze student academic performance. The system allows users to enter and store marks for multiple students, and then performs basic statistical analysis on the collected data. It calculates important metrics such as the average marks, highest marks, and lowest marks, helping users gain insights into overall student performance.

The application features a simple text-based menu that enables users to interact with the system easily. Developed using fundamental C++ concepts such as arrays/vectors, functions, loops, conditionals, and user input/output, this project serves as a practical implementation of core programming principles while demonstrating how data can be processed and analyzed efficiently.
